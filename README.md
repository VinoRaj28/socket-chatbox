# 🔒 Socket Chat App

A real-time multi-client chat application built with Python WebSockets and a browser frontend.
Built as part of my Cybersecurity & Networking learning journey.

## 🛠️ Tech Stack
- Python 3 — `socket`, `websockets`, `asyncio`
- HTML, CSS, JavaScript (vanilla)
- Linux + VS Code

  ## 🔍 What I Learned
- How TCP/IP client-server architecture works
- How raw data travels over a network as bytes
- Why browsers need WebSockets instead of raw TCP
- Multi-client broadcasting with asyncio
- Why plaintext chat is vulnerable (Wireshark demo)
